# Banco2026v Fullstack

Incluye:
- backend Spring Boot + Maven + MySQL
- frontend React + Vite

## Backend
1. Crear o levantar MySQL
2. Ajustar `backend/src/main/resources/application.properties` si necesitas otra contraseña
3. Ejecutar:

```bash
cd backend
mvn spring-boot:run
```

## Frontend
```bash
cd frontend
npm install
npm run dev
```

## Rutas mínimas cubiertas
- Consultar clientes
- Realizar transferencia
- Historial por cliente

## Funcionalidad opcional incluida
- Crear cliente
- Actualizar cliente
- Borrar cliente
